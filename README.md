# checklists
Rey W Project

A checklist/to-do app that uses tableViewControllers.  

**Current Features**  
-Add from nav bar  
-Edit from detail disclosure  
-Checkmark on tap
-Delete on swipe  
-Data Persistence with dataModel class
-Add lists of items (ex. Birthdays, Groceries, To Do)
